Ergo-E

This is an example of a simple, professional web presence. Further personalization can be completed using staff portraits, company logos, and a customized color scheme. The point of this project is to convey website layout and the application of company information. The landing page has many sections to convey the message of the company in a refined but interesting way, with the headline slider adding some flair to the page. The contact page is designed to supply a lot of information in a streamlined way using forms, maps, and icons. Multiple call to action sections on the website announce the company is hiring, and can be used to display important information.
